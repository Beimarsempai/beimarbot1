## Instale las dependencias
```bash
> pkg upgrade & update
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick

```
## Clona este proyecto
```bash
> git clone https://github.com/Fxmods/FX-70

> cd FX-70
```

## Instala el bot
Antes de ejecutar el siguiente comando, asegúrese de estar en el directorio del proyecto que
acabas de clonar !!

```bash

> bash install.sh

### Inicia el bot

> npm start
```
